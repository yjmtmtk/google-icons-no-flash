Google Icons No Flash Demo

Testing FOUT (Flash of Unstyled Text) Prevention:

  1. Open Network tab in DevTools and set Throttling to "Fast 3G".
  2. Hard Reload (Cmd+Shift+R).
  3. Notice that icons don't appear until they've fully loaded. You will never see raw text like "home" or "search".
